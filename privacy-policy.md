# ClickStyle — Privacy Policy

**Last updated:** March 17, 2026

## Overview

ClickStyle is a Chrome extension that allows users to edit CSS styles of any element on a webpage. This privacy policy explains how ClickStyle handles user data.

## Data Collection

**ClickStyle does not collect, store, or transmit any user data.** Specifically:

- No personal information is collected
- No browsing history is tracked
- No website content is read, stored, or sent to any server
- No cookies are set
- No analytics or tracking scripts are used
- No third-party services are integrated
- No user accounts are required

## How It Works

All functionality runs entirely within your browser. When you use ClickStyle to edit an element's styles, those edits exist only in your current browser session. Nothing is saved or sent anywhere. Refreshing the page removes all edits.

## Permissions

ClickStyle requests the following browser permissions:

- **activeTab** — To access the current page and modify element styles when you interact with them
- **scripting** — To inject the editor interface into the active page
- **contextMenus** — To add a right-click menu option for activating the editor

These permissions are used solely to provide the core editing functionality. No data is accessed beyond what is needed to display and modify styles on the page you are viewing.

## Remote Code

ClickStyle does not load or execute any remote code. All code is bundled within the extension package.
